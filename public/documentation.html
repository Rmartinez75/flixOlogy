flixOlogy Documentation

The flixOlogy REST API, using Express.js, holds and shares the description, genre and director of each movie along with user information:

/movies - GET - all movies and information

/movies/title - GET - sends back information of a specific movie by title

/directors - GET - the name of all directors in the directors collection

/director/Name - GET - sends back information of specified director by name

/genres - GET - sends back a list of all genres

/genres - GET - sends back information about specific genre by name

/users - POST - register a new user and create an account

/users/Username - PUT - update user information

/users/Username/MoviesID - PUT - user can add a specified movie to favorites list

/users/Username/favoritesMoviesID - DELETE - users can delete a movie from favorites list

/users - DELETE - user can delete their account
