flixOlogy Documentation

The flixOlogy REST API, using Express.js, holds and shares the information of movies along with actors, cast and crew using 12 endpoints:

/movies - GET - all movies titles

/movies/info - GET - detailed information about the movies

/movies/info/genre - GET - the genres of the movies

/castCrew/director/[name] - GET - the name of specified director

/users - POST - register a new user and create an account

/users - PUT - update user information

/users/[name]/favorites - PUT - user can add a specified movie to favorites list

/users/favorites - DELETE - users can delete a movie from favorites list

/users - DELETE - user can delete their account

/castCrew/name - GET - actor information and their resume of movies

/movies/info/[actors] - GET - list of actors starring in specified movies

/users/[name]/favorites/watchList - PUT - adds a watch list to user favorites list

All this data is returned as json format like so: { "movieName:" "Sunshine" }
