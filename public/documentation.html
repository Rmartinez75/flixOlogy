flixOlogy Documentation

The flixOlogy REST API, using Express.js, holds and shares the information of movies along with actors, cast and crew using 12 endpoints:

/movies - GET - all movies and information

/movies/genre - GET - the genres of the movies

/movies/director/Name - GET - the name of specified director

/users - POST - register a new user and create an account

/users - PUT - update user information

/users/username/favoritesMovies - PUT - user can add a specified movie to favorites list

/users/favoritesMovies - DELETE - users can delete a movie from favorites list

/users - DELETE - user can delete their account

All this data is returned as json format like so: { "movieName:" "Sunshine" }
